Step by step guide 
1. Install all the packages which is in requirement.txt file 
2. now change the folder structure accordingly to store the pdf files for analysis
3. run the Engaigemodelling.py file to create a metadata.json and vector_index.json
4. update the query as per the requirement in EngaigeQuery File and also top how many result you are expecting in top_k
5. run the EngaigeQuery.py file and check the result. cross check the same using files and see if it is relevant to your question or not.
